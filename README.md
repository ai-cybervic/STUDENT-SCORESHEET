# Student Records Management Program

This Java program is designed to manage student records for a class. It allows users to input student information, calculate total marks, assign ranks, and display a summary table with averages and rank counts.

---

## Features

- Input basic school information:
  - School Name
  - Teacher's Name
  - Grade
  - Year
- Record multiple studentsâ€™ data:
  - Student names
  - Marks in English, Math, History, Geography, Science, and Programming
- Calculate total marks for each student
- Assign ranks based on total marks:
  - `A` for >= 540
  - `B` for >= 480
  - `C` for >= 420
  - `D` for >= 360
  - `F` for below 360
- Calculate totals and averages for each subject
- Display a neatly formatted table of student records
- Summarize rank counts

---

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed (version 8 or higher)
- A text editor or IDE (e.g., VS Code, IntelliJ IDEA, Eclipse)

### Running the Program

1. Save the code as `Main.java`.
2. Open a terminal or command prompt in the directory containing the file.
3. Compile the program:
   ```bash
   javac Main.java
```
